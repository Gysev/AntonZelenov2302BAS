## Лабораторная работа №3

В данной работе был реализован REST API сервиса доставки на Spring Boot.

В проекте описаны основные сущности предметной области: отправитель, получатель, курьер, посылка и доставка.
Для каждой сущности реализованы стандартные CRUD-операции через HTTP-запросы.

Помимо базового CRUD была добавлена бизнес-логика сервиса доставки.
Реализованы сценарии создания заказа (создание посылки и доставки за одну операцию), назначения курьера на доставку, изменения статуса доставки, завершения доставки и получения списка доставок конкретного курьера.

Для бизнес-операций используются отдельные DTO-классы, что позволяет отделить логику API от внутренних моделей данных.

Работа сервиса протестирована с помощью Postman. Создана коллекция запросов, демонстрирующая полный сценарий работы: от создания данных до выполнения бизнес-операций.
